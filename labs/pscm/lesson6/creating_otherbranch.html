Creating another branch

Let’s go to the branch explorer and check the current status:

Figure 1: Branch explorer

You are in SCM001 branch. Let’s return to main branch. Right click on the branch and click on “Switch workspace to this changeset”. It looks like:

Figure 2: Switch workspace to this changeset

After clicking you confirm the operation and you will see something similar this:

Figure 3: Current changeset at main branch

The current changeset has come back to the main branch. Now you’re goint to create another branch from the last changeset on the main brnach. These are the steps to follow:

  • Right click on the current changeset, and click “Create branch from this changeset”
  • Write a branch name, for example:”SCM002 (for task002)
  • Click OK

Now your branch explorer will look like the following image:

Figure 4: Branch Explorer
