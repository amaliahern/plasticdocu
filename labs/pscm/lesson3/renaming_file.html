Renaming a file

Renaming a file is actually equivalent to a moving it as you did before, but a rename is just a move inside the same directory.

You can select a file and press ‘F2’ or right click to open up the context menu and select ‘rename’ as the figure below shows.

Figure 1: Renaming a file

After clicking “rename” a dialog will show up and will let you introduce a new name for the selected item:

Figure 2: Windows Dialog

For example, we can write “readme_dokan.txt” as new name. Once the operation completes, the “Pending Changes View” will look like the following:

Figure 3: Pending Changes View

Please note the status of the file is now “Moved”. Plastic knows the file has been renamed. Introduce a comment and checkin.
