Deleting a file

The next simple operation will be deleting a file under the Plastic SCM control.

In order to do that, right click on a “readme.ja.txt” file and click on “delete” as the figure below shows:

Figure 1: Deleting a file

After clicking “delete” a dialog will show up to select whether you want to keep the file on disk or not.

Figure 2: Windows Dialog

You’ll choose ‘do not delete items on disk’ when you just want to remove something from version control but you still want to keep it (like a temporary file that you mistakenly added).

Choose now “Do not delete the items on disk”. The pending changes will look like the following:

Figure 3: Pending Changes View

Go now to the “Items View”. Please note the status of the deleted file is now “private” as the image below shows:

Figure 4: Items View

It means that file is not under source control anymore. The “Items View” displays private file without customized icon and no entry in Type, Branch, and Repository columns.

Figure 5: Items View: Private Files
