Plastic SCM Labs

Welcome to Plastic SCM labs. Here you will find a collection of labs to help you understand Plastic SCM, using GUI tool.

Each lab explains the steps required to solve a specific problem.

This lessons uses a tutorial format that incrementally works on a project that teaches you some fundamental concepts about distributed version control system. So it's important you follow each step.

Before you start these lessons, be sure you have your development environment set up. You need to:

Download Plastic SCM
Visit download page
Install Plastic SCM
Launch the installer to begin installation wizard
Set up Plastic SCM
Select the hostname and port where you Plastic SCM server is located
