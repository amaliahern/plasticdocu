Creating a “baseline”

What’s a baseline? It’s very simple: a label. It’s a snapshot of your code at a given point in time. It’s a mark, a tag you can use to rebuild this status later.

In order to create a label, open the “labels view” by clicking on Labels icon, inside the section Other actions in the left panel. Your Label View will look like this:

Figure 1: Creating a baseline

Click on “Create new label”. After clicking a dialog will show up to introduce the name for the new label:

Figure 2: New label

Please, note “the changeset to label” field. It tells Plastic where the label is applied. The default value is the current changeset loaded in the workspace. In my case, the changeset is number 6.

For example, enter “Baseline002” for the name and enter a comment as the following figure shows:
