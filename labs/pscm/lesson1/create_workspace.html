Create a workspace

Now let’s create a workspace on the location where you uncompressed the files (remember, in my case it was C:\Users\pablo\dokan_workspace).

In order to do it, go to the actions bar on the right and then click on “workspaces”:

Figure 1: Workspaces

Then the “workspaces view” will show up, listing all the available workspaces on your machine. Click on the “create workspace” button:

Figure 2: Create new workspace

The “create workspace” dialog will show up and then you’ve to give a name to the workspace, a path and select a repository that the workspace will use:

Figure 3: New workspace

As you can see on the figure, I chose “dokan_workspace” as name, then I wrote the path I used to place the code downloaded from the web, then I selected the repository I’ve just created.

As you can see, there’s a new tab on the workspaces tab, with the name of the recently created workspace:

Figure 4: Workspace created
